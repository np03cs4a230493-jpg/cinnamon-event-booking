# ☕ Cinnamon & Co. Event Booking System

A full-stack MERN application for managing cafe events, ticket sales, and community engagement.

## 🚀 Features
- **User Side:** Browse events, book tickets, and suggest new event ideas.
- **Admin Dashboard:** Manage events (Create/Delete), view sales analytics (Revenue/Tickets Sold), and moderate community suggestions.
- **Authentication:** Secure login/signup with role-based access (User vs. Admin).
- **Image Uploads:** Dynamic image handling for event posters.

## 🛠️ Tech Stack
- **Frontend:** React.js, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Styling:** Custom CSS / Modern UI
